# Academic Analytics for Institutional Decision Making

## Project Description

The **Academic Analytics for Institutional Decision Making** project focuses on analyzing student academic performance data using Microsoft Excel and Artificial Intelligence tools to generate meaningful insights for academic decision-making. This project transforms raw student data into actionable insights that help institutions identify student performance trends, subject difficulty levels, attendance impact, and department-wise performance.

This project was developed as part of the **AICTE Edunet – AI Careers for Women (AICW) Fellowship 2026**.

---

## Problem Statement

Educational institutions collect large amounts of student data such as marks, attendance, and grades every semester. However, this data is often not analyzed effectively, making it difficult for faculty and management to identify struggling students, compare departmental performance, and make data-driven decisions. This project aims to solve this problem by building a structured Excel dashboard and generating AI-based insights from the data.

---

## Objectives of the Project

* To collect and understand the student dataset.
* To clean and preprocess the dataset.
* To analyze student performance using Excel formulas and Pivot Tables.
* To calculate Key Performance Indicators (KPIs).
* To build an interactive Excel dashboard.
* To generate AI-based insights using AI tools.
* To provide recommendations for academic improvement.

---

## Dataset Information

The dataset contains **1000 student records** from four engineering branches:

* CSE
* ECE
* MECH
* CIVIL

### Dataset Fields

* Student ID
* Gender
* Branch
* Section
* Attendance Percentage
* Mathematics Marks
* Physics Marks
* Chemistry Marks
* Average Score
* Grade
* Result (Pass/Fail)
* Parental Education Level

## Data Analysis Process

1. Data Collection – Student dataset created.
2. Data Cleaning – Blank rows removed, values standardized, data validated.
3. Data Analysis – Pivot Tables and Excel formulas used.
4. KPI Calculation – Pass rate, average score, attendance analysis, grade distribution.
5. Dashboard Creation – Charts and KPI cards created in Excel.
6. AI Insights – AI used to generate insights and summary.
7. Report Generation – Project documentation prepared.

## Key Performance Indicators (KPIs)

* Total Students
* Average Score
* Overall Pass Rate
* Total Fail Students
* Average Attendance
* Grade Distribution
* Branch-wise Performance
* Subject-wise Performance
* Attendance vs Performance

---

## Key Insights

* The overall pass rate is **96.7%**, indicating strong academic performance.
* **MECH** branch has the highest average score.
* **ECE** branch has the highest pass rate.
* **Mathematics** is the most difficult subject based on average marks.
* Students with **low attendance** are more likely to fail.
* Students with **high attendance** perform better academically.
* Female students performed slightly better than male students.
* Parental education level influences student performance.
